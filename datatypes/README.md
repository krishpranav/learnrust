## Basic DataTypes:

- main.rs

- names
```rust
fn main() {
    /* declaring name */
    let name = "User One";

    /* printing it out */
    println!("name: {}", name);
}
```

- numbers (int)
```rust
fn main() {
    /* for storing or declaring age use u32 */
    let age:u32 = 20; 

    /* printing the data that we stored */
    println!("age: {} ", age);
}
```
